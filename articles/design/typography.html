Типографика.

Типографика — техника в дизайне, которая описывает работу с текстом. Текст составляет большую часть любого интерфейса, поэтому от типографики зависит, как пользователь будет воспринимать дизайн.

Параметры текста.

  • Гарнитура — набор шрифтов, отличных по начертанию, но одинаковых по характеру изображения букв.
  • Шрифт — одно конкретное начертание в рамках гарнитуры.
  • Размер шрифта.
  • Трекинг — расстояние между всеми символами в границах одного слова.
  • Кернинг — корректировка расстояний между парами букв. Без кернинга текст выглядел бы неоднородным, ввиду того, что символы имеют разный визуальный вес.
  • Межстрочное расстояние (интерлиньяж).

Правила.

Шрифтовые пары.

Шрифты должны сочетаться, дополнять друг друга.

Шрифтовая пара — это два контрастных (явно различающихся) шрифта, при этом сочетающиеся между собой.

Самый простой пример шрифтовой пары — заголовок в начертании Bold и наборный текст в начертании Regular.

Вид гарнитуры.

Большие объемы текста, набранные антиква (с засечками) читаются легче.

Гротеск (без засечек) чаще используют для заголовков и небольших текстов.

Декоративные используются только для украшения (в заголовках, логотипах).

Вид шрифта.

Жирные и курсивные шрифты не стоит использовать для наборного текста, поскольку они плохо читаются. Они подойдут для заголовков и коротких текстовых элементов.

Количество шрифтов.

В проекте должно быть не больше двух шрифтов.

Размер шрифта.

Для основного текста используется размер шрифта не менее 16px — текст с таким размером хорошо читается на всех устройствах. Для незначительных текстовых элементов можно использовать 14px, но ниже этой границы уходить не стоит.

Трекинг.

Прописные буквы необходимо разрежать всегда — увеличивать трекинг, а строчные буквы — никогда.

Межстрочное расстояние.

Чем длиннее строка, тем больше межстрочное расстояние, и наоборот.

Обычно используют значение между 120–150% от размера шрифта.

Длина строки.

Оптимально размещать в строке около 60 символов. Тогда пользователь не почувствует дискомфорт при чтении текста.

На мобильных устройствах используйте строки из 30–40 символов.

Базовая линия.

Базовая линия (baseline) — невидимая линия, проходящая по нижнему краю букв без учёта выносных элементов.

Текстовые элементы с разным размером шрифта должны выравниваться по базовой линии.

Использование стилей.

Не используйте много стилей в одном текстовом элементе. Это выглядит плохо, каждый элемент будет пытаться перетянуть внимание на себя.

Типографические мелочи.

Висячие предлоги лучше убрать.

Ставьте правильные дефисы, короткое тире и тире. Дефис используется в пределах одного слова: как-то, где-нибудь. Короткое тире ставят между датами и числами: 1992–1998, январь–февраль. Тире используется между словами в предложении: арбуз — это ягода.

Кавычки. В русском языке используются ёлочки — «». Если кавычки в кавычках, то внутри ёлочек используются лапки — „“.

Не забывайте про буквы «ё» и «й». Они тоже существуют, и их нужно использовать.

Орфография и пунктуация. Всегда проверяйте текст на ошибки. Иногда проще попросить об этом другого человека, поскольку при долгой работе с текстом глаз замыливается.
