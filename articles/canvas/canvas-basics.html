Основы Canvas

Тег <canvas></canvas> используется для рисования графики средствами языков программирования (обычно это JavaScript). Он может, к примеру, использоваться для рисования графиков, создания коллажей и анимаций.

Этот тег имеет два атрибута: width и height, в которых можно задать его размеры.

<canvas width="320" height="200"></canvas>

По умолчанию <canvas></canvas> абсолютно прозрачный и имеет размеры 300px × 150px.

<canvas></canvas> может быть стилизован (margin, border, background и т. д.), но это не влияет на рисунок.

Внутрь тега может быть вложено альтернативное содержимое, которое будет отображено, если браузер не поддерживает <canvas></canvas>.

Проверить поддерживает ли бразер <canvas></canvas> можно из кода, проверив наличие метода getContext:

const canvas = document.querySelector('#my-canvas');
if (canvas.getContext) {
  // ...
}

Содержимое Canvas недоступно для программ чтения с экрана. Если холст является чисто декоративным, нужно добавить атрибут role="presentation". В противном случае можно добавить описание в атрибуте aria-label.

Рисование в 2D

Для начала нужно найти элемент и получить контекст:

const canvas = document.querySelector('#my-canvas');
const ctx = canvas.getContext('2d');

Контекст предоставляет свойства и методы для рисования.

Пример:

ctx.fillStyle = 'rgb(200, 0, 0)';
ctx.fillRect(10, 10, 50, 50);

ctx.fillStyle = 'rgba(0, 0, 200, 0.5)';
ctx.fillRect(30, 30, 50, 50);
