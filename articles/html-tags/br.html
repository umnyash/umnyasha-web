Тег br.

Тег <br> (сокращение от «line break») добавляет перенос строки в тексте для повышения его читабельности, например, в почтовых адресах, стихах, текстах песен, режиме работы.

Иногда этот тег применяют неправильно, для разбиения текста на «как бы абзацы». Никогда так не делайте. Для разметки абзацев нужно использовать тег <p></p>.
