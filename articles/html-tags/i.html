Тег i.

Тег <i></i> применяется для обозначения текста, который отличается от окружающего его текста по качеству. Например, в <i></i> можно заключать технические термины, иностранные слова, транслитерации. Также в этот тег можно обернуть мысли героя. В речи такой текст обычно выделяется интонационно.

Пример:

Он взглянул в окно и подумал — <i>такого просто не может быть</i>!

Слова на языках, отличающихся от основного текста, должны иметь атрибут lang.

По умолчанию тег <i></i> выделяется курсивом.
