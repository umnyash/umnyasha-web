Тег nav.

Для создания логического раздела с основной навигацией предназначен тег <nav></nav> (сокращение от английского «navigation»). Обычно в <nav></nav> включают ссылки на другие страницы или навигацию по текущей странице.

Кроме того, блок <nav></nav> помимо ссылок может включать абзацы с текстом, заголовки, списки и другое содержание.

Не каждая группа ссылок на странице должна быть обёрнута в <nav></nav>. Например, небольшой блок со вспомогательными ссылками в подвале сайта. Такой блок внутри тега <footer></footer> не нужно дополнительно оборачивать в тег <nav></nav>.

В случае, когда навигация шапки дублируется в подвале сайта, не рекомендуется использовать тег <nav></nav> повторно. Достаточно будет использовать список со ссылками.
