Тег meter

Тег <meter></meter> предназначен для визуализации тех шкал, которые не относятся к прогрессу. Например, использованное дисковое пространство, доля голосов населения за определённого кандидата и т.д.

Атрибуты:

  • value — текущее значение.
  • min — нижняя граница диапазона.
  • max — верхняя граница диапазона.
  • low — значения ниже считаются низкими.
  • high — значения выше считаются высокими.
  • optimum — оптимальное значение.

По умолчанию низкие и высокие значения делают полоску жёлтой, при остальных значениях она отображается зелёной.

По умолчанию минимальное значение равно 0, а максимальное 1.

Пример:

Использование диска: <meter min="0" value="164" max="500">Использовано 164Гб из доступных 500Гб</meter>
