Тег video

Тег <video></video> предназначен для вставки видео. Но не всех, а только тех, на которые мы можно дать прямую ссылку. Зачастую это видеофайлы, которые хранятся на нашем собственном сервере. Это материалы, за которые мы переживаем: уникальный контент, который нам не хочется выкладывать на внешние сервисы вроде YouTube.

Основные атрибуты

  • width и height — ширина и высота видео.
  • src — адрес видеофайла.
  • controls — добавляет элементы управления видеоплеером.
  • poster — адрес картинки-обложки, которая отображается когда видео ещё не загрузилось или не воспроизводится.
  • autoplay — при наличии этого атрибута воспроизведение видео начинается автоматически.
  • preload — режим предзагрузки видео.

Форматы и источники видео

Адрес видеофайла можно указать в атрибуте src тега <video></video>, но...

На текущий момент существует несколько форматов видео, каждый из которых хорошо поддерживается лишь несколькими браузерами.

Самые распространённые форматы:

  • MPEG-4/H.264.
  • OGG/Theora.
  • WebM.

Поэтому необходимо использовать видео во всех этих форматах, а их адреса указать с помощью тегов <source>: в атрибуте src указывается адрес видеофайла, а в атрибуте type его тип (также там могут указываться и кодеки).

<video controls>
  <source src="video.mp4" type="video/mp4">
  <source src="video.ogv" type="video/ogg">
  <source src="video.webm" type="video/webm">
</video>

Браузер из списка видеофайлов выбирает первый, который может проиграть и загружает его.

Атрибут type не является обязательным, так как браузер умеет сам определять тип и кодеки, но указывая тип явно, мы помогаем ему не ошибиться.

На YouTube невозможно получить прямую ссылку на видеоролик, чтобы вставить его в тег <video></video>, и это приходится делать с помощью <iframe></iframe> — тега для вставки фреймов.
