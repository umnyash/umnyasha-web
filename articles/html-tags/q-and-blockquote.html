Цитаты.

В HTML существует 2 тега для разметки цитат.

Тег q, встроенная цитата.

Тег <q></q> (сокращение от «quote») предназначен для выделения цитат внутри предложения. Текст внутри этого тега браузер автоматически обрамляет кавычками, поэтому добавлять кавычки вручную не нужно.

Пример:

Как писал Бернард Корнуэлл в <cite>Саксонских хрониках</cite>, <q>Судьба неумолима</q>.

Если известен интернет-адрес цитаты, то его можно указать в атрибуте cite:

В спецификации говорится: <q cite="https://www.w3.org/TR/html52">The q element represents some phrasing content quoted from another source</q>.

С помощью CSS-свойства quotes, можно задать другие кавычки для обрамления цитаты. Для этого в значении свойства записываются Unicode-символы (каждый в кавычках) для открывающей и закрывающей кавычек.

Пример:

q {
  quotes: "„" "“";
}

Тег blockquote, отдельная цитата.

Для выделения длинных цитат, которые могут состоять из нескольких абзацев, предназначен тег <blockquote></blockquote>. Он выделяет цитату не как фрагмент текста в предложении, а как отдельный блок текста с отступами.

Пример:

<blockquote>
  Рыба очень вкусная!
</blockquote>

Авторство цитаты, при наличии, должно размещаться за пределами <blockquote></blockquote>:

<blockquote>
  Рыба очень вкусная!
</blockquote>
<p>Русик</p>

Так же, как и у <q></q>, у <blockquote></blockquote> может быть атрибут cite, содержащий ссылку на источник цитаты. Этот атрибут нужен для соблюдения авторского права.
