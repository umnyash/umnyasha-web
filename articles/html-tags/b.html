Тег b.

Тег <b></b> (сокращение от «bold») предназначен для выделения текста с целью привлечения к нему внимания, но без придания ему особой важности. Например, для выделения вводного предложения статьи или ключевых слов.

По умолчанию текст выделяется жирным.
