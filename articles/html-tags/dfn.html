Тег dfn

Тег <dfn></dfn> определяет термин, при этом непосредственный родитель этого тега (например, параграф), также должен относиться к термину и расшифровывать его значение.

Обычно используется в сочетании с <abbr></abbr> — тегом для аббревиатур. Для этих тегов атрибут title выступает в значении «расшифровка», а id может связывать ссылкой якорем с расшифровкой внутри материала.

Пример:

<p>Как мы уже знаем, <dfn id="html"><abbr title="HyperText Markup Language">HTML</abbr></dfn> — это язык для вёрстки веб-страниц.</p>

<p>Используя язык <a href="#html"><abbr title="HyperText Markup Language">HTML</abbr></a> мы формируем содержание и структуру сайтов.</p>
