Тег code

С помощью тега <code></code> размечается любой фрагмент текста, который распознаётся компьютером: код программы, разметки, названия файлов и другое. Обычно браузеры отображают текст в теге <code></code> моноширинным шрифтом.

Пример:

Тег <code>&lt;ul&gt;&lt;/ul&gt;<code> — это неупорядоченный список.

Какого-то специального способа указать на каком языке написан фрагмент кода, у тега <code></code> нет.

Тег <code></code> можно вкладывать внутрь тега <pre></pre>, а наоборот делать нельзя.

<pre><code>
  if (a > b) {
    console.log("Hello!");
  }
</code></pre>
