Теги img и picture

Тег <img> предназначен для подключения контентых изображений. Внутри него с помощью атрибута src (сокращение от «source»), необходимо написать адрес, ведущий к картинке.

<img src="img/logo.svg">

Размеры изображения

По умолчанию размеры изображения равны его исходным значениям.

В HTML, чтобы управлять шириной или высотой изображения, нужно использовать атрибуты width и height. Размеры в этих атрибутах задаются без единиц измерения.

<img src="img/logo.svg" width="200" height="100">

В примере выше изображению задана ширина 200px и высота 100px.

Если задать только один из размеров, ширину или высоту, то вторую размерность браузер вычислит самостоятельно исходя из пропорций изображения.

При работе с высотой и шириной изображения нужно быть очень аккуратным. Ведь если задать картинке одновременно и высоту и ширину, то браузер может нарушить пропорции исходного изображения.

Атрибут alt

Чтобы было понятно, что изображено на картинке, в случаях, когда она не может отобразиться, существует альтернативный текст. Ещё альтернативный текст помогает сайтам оставаться доступными, например, для категории пользователей, которая не имеет возможности видеть картинки.

Альтернативный текст задаётся с помощью атрибута alt. Например:

<img src="files/cat.jpg" alt="Небольшой серый кот.">

Максимальная рекомендуемая длина – 125 символов.

Адаптивные изображения

Тег <img> может подключать разные изображения в зависимости от пиксельной плотности экрана.

Для этого в атрибуте srcset нужно указать путь и дескриптор x, обозначающий для экранов какой плотности предназначено изображение.

В примере добавлены версии изображения для 2x, 3x и 4x экранов:

<img src="files/cat.jpg" srcset="files/cat@2x.jpg 2x, files/cat@3x.jpg 3x, files/cat@4x.jpg 4x" alt="Небольшой серый кот.">

Кроме этого тег <img> может подлючать разные изображения в зависимости от нужной ширины.

Для этого в атрибуте sizes нужно задать ширину изображения, а в атрибуте srcset перечислить пути и дескрипторы w, обозначающие реальные значения ширины изображений.

В примере задана ширина 50vw – половина экрана, и пути к изображениям с шириной 400px и 800px:

<img src="files/cat-200.jpg" sizes="50vw" srcset="files/cat-400.jpg 400w, files/cat-800.jpg 800w" alt="Небольшой серый кот.">

Таким образом, если ширина экрана 320px, то 50vw будет равняться 160px. В этом случае загрузиться изображение cat-200.jpg, так как это самый оптимальный вариант. Если ширина экрана будет равна 768px, то половина экрана будет равна 383px и будет загружено cat-400.jpg. При этом учитывается пиксельная плотность экрана. Если плотность будет равна 1.5, то при ширине 768px требуемая ширина изображения станет (768px * 1.5) / 2 = 576px и будет использовано cat-800.jpg.

В атрибуте sizes может быть перечислено несколько размеров (дополнительные размеры имеет смысл использовать только с условиями). Среди них будет выбран первый подходящий вариант.

В примере ниже на ширине до 399px включительно размер будет равен 100vw, на большей ширине – 400px:

<img src="files/cat-200.jpg" sizes="(max-width: 399px) 100vw, 400px" srcset="files/cat-400.jpg 400w" alt="Небольшой серый кот.">

Размеры в атрибуте sizes нельзя указывать в процентах.

Если при использовании srcset с дескриптором w не задан атрибут sizes, то размером для подбора изображения будет считаться 100vw.

Тег picture

Тег <picture></picture> должен содержать один тег <img> и может содержать теги <source>, которые предоставляют дополнильные возможности для выбора подключаемого изображения.

Например, использовать медиавыражения для подлючения разных изображений:

<picture>
  <source media="(min-width: 768px)" srcset="cat-tablet.jpg, cat-tablet@2x.jpg 2x">
  <img src="files/cat-mobile.jpg" srcset="cat-mobile@2x.jpg 2x" alt="Небольшой серый кот.">
<picture/>

или подключать изображения определённого типа. В примере ниже, если браузер поддерживает форма WebP, то будет использовано изображение в этом формате вместо JPG.

<picture>
  <source type="image/webp" srcset="files/cat.webp, files/cat@2x.webp 2x">
  <img src="files/cat.jpg" srcset="files/cat@2x.jpg 2x" alt="Небольшой серый кот.">
<picture/>
