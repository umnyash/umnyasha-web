Тег audio.

Тег <audio></audio> предназначен для вставки аудио.

Основные атрибуты.

  • src — адрес аудиофайла.
  • controls — добавляет панель управления проигрывателем.
  • autoplay — при наличии этого атрибута воспроизведение аудио начинается автоматически.
  • preload — режим предзагрузки аудио.

Форматы и источники аудио.

Для охвата большинства современных браузеров, достаточно использовать всего 2 формата:

  • MP3.
  • OGG.

Поскольку в атрибуте src тега <audio></audio> можно указать адрес только одного аудиофайла, необходимо использовать теги <source>.

<audio controls>
  <source src="sound.mp3" type="audio/mpeg">
  <source src="sound.oga" type="audio/ogg">
</audio>
