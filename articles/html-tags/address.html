Теги address

Тег <address></address> представляет собой контактную информацию, относящуюся ко всей странице <body></body>. Если тег расположен внутри <article></article>, то контактная информация относится непосредственно к этой статье.

Этот тег должен содержать только реальный метод связи: ссылку на автора, электронную почту, номер телефона, почтовый адрес.

Пример:

<address>
  Для получения подробной информации свяжитесь с
  <a href="email@mail.ru">Умняшем</a>
</address>
