Тег progress.

Тег <progress></progress> предназначен для визуализации прогресса выполнения задачи. Имеет два основных атрибута: value — текущее значение, и max — максимально возможное значение. Если максимальное значение не задано, текущее можно указывать в виде десятичной дроби от 0 до 1.

Пример:

Прогресс: <progress max="100" value="10">10%</progress>

Если атрибут value не задан, по умолчанию полоска прогресса будет перемещаться от одного края к другому, показывая, что система ожидает данных.
