Медиа-запросы

Медиа-запросы позволяют применить стили в зависимости от характеристик браузера.

Синтаксис:

@media (условие) {
  ...
}

Характеристики

Современные устройства имеют множество характеристик.

Примеры характеристик:

  • min-resolution: 2dppx — минимальная плотность пикселей от 2dppx
  • min-width: 1024px — минимальная ширина вьюпорта 1024px
  • width: 1024px — ширина вьюпорта 1024px
  • max-width: 1024px — максимальная ширина вьюпорта 1024px
  • min-height: 1024px — минимальная высота вьюпорта 1024px
  • height: 1024px — высота вьюпорта 1024px
  • max-height: 1024px — максимальная высота вьюпорта 1024px

Операторы

Медиавыражения поддерживают логические операторы and и or.

Пример использования and:

@media (min-width: 768px) and (max-width: 1024px) {
  ...
}

Запись условий через запятую равнозначна ипользованию or:

@media (max-width: 1024ppx), (min-resolution: 2dppx) {
  ...
}
