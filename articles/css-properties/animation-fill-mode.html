Свойство animation-fill-mode

Свойство animation-fill-mode задаёт режим применения стилей до и после анимации.

Значения:

  • none — (по умолчанию) не применять. После завершения анимации элемент вернётся в исходное состояние.
  • backwards — применить стили первого кадра до начала анимации. Это заметно, если анимация имеет задержку.
  • forwards — применить стили последнего кадра по завершении анимации.
  • both — совмещает эффект значений backwards и forwards.
