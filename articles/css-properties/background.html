Свойство background.

Свойство background параметры фона.

Составное свойство:

  • background-color — цвет.
  • background-image — изображение.
  • background-repeat — повтор.
  • background-attachment — привязка.
  • background-origin — позиционирование.
  • background-clip — обрезка.
  • background-position — позиция.
  • background-size — размер.

Значения:

  • Параметры фона, перечисленные через пробел.

Если какой-то параметр не указан, то берётся значение по умолчанию.

Если в свойстве задана позиция фона и его размер, то эти значения пишутся в самом конце через слеш (/).

Примеры:

background: #e74c3c;
background: url("image.png") no-repeat;
background: url("image.png") no-repeat 10px 20px;
background: url("image.png") 10px 20px / 300px 300px;
=^-^=
