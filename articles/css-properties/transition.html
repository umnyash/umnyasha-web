Свойство transition.

Свойство transition задаёт параметры перехода.

Составное свойство:

  • transition-property — свойство.
  • transition-duration — длительность.
  • transition-timing-function — темп.
  • transition-delay — задержка.

Значения:

  • Перечисленные через пробел параметры для одного или нескольких переходов, перечисленных через запятую.

Пример:

transition: width 1s ease-in 2s;
transition: width 1s ease-in 2s, height 2s ease-out 1s;
