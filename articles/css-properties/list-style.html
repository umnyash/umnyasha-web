Свойство list-style.

Свойство list-style задаёт стили маркеров списка.

Составное свойство:

  • list-style-type — вид маркера или номера.
  • list-style-image — картинка, которая будет отображаться вместо маркера.
  • list-style-position — позиция маркера, относительно элемента списка.

Значения:

  • Перечисленные через пробел в любом порядке значения стилей.
  • none — удаление маркеров.

Пример:

list-style: upper-roman inside url("rocket.svg");
list-style: none;
