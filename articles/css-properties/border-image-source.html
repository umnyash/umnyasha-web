Свойство border-image-source

Свойство border-image-source задаёт фоновое изображение рамки.

Значения:

  • Адрес изображения внутри url("...").

Пример:

border-image-source: url("img/sky.jpg");

По умолчанию изображением заполняются только углы рамки. Изменить это можно с помощью свойства border-image-slice.
