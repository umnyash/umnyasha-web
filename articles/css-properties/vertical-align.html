Свойство vertical-align.

Свойство vertical-align задаёт вертикальное выравнивание инлайновых элементов относительно строки.

Значения:

  • Ключевое слово
  • Положительное или отрицательное значение в px или %.

Ключевые слова:

  • baseline — (по умолчанию) выравнивание по базовой линии строки.
  • top — выравнивание по верхнему краю строки.
  • middle — выравнивание по середине строки.
  • bottom — выравнивание по нижнему краю строки.
  • sub — опускает базовую линию блока вниз для создания нижнего индекса.
  • super — поднимает базовую линию блока вверх для создания верхнего индекса.
  • text-bottom — нижняя граница элемента выравнивается по нижнему краю содержимого родителя.
  • text-top — верхняя граница элемента выравнивается по верхему краю содержимого родителя.

Значения в % рассчитываются от высоты строки элемента (line-height).

Значение 0 эквивалентно baseline.

В отличие от text-align свойство vertical-align задаётся самому элементу, а не содержащему его контейнеру.

Отступ под картинкой.

При вставке изображения под картинкой остаётся небольшой отступ, который часто нужно убрать.

Откуда берётся этот отступ? Тег <img> обладая строчным боксом, по умолчанию также выравнивается по базовой линии, и этот отступ является пространством под выносной элемент букв (например, под хвостики букв «р» или «у»).

Избавиться от этого отступа можно сменив тип бокса или способ вертикального выравнивания.
