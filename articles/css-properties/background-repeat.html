Свойство background-repeat

Свойство background-repeat задаёт режим повтора фонового изображения.

Значения:

  • repeat — (по умолчанию) повтор во всех направлениях.
  • repeat-x — повтор только по горизонтали.
  • repeat-y — повтор только по вертикали.
  • no-repeat — не повторять.
  • space — повторять так, чтобы в области поместилось целое число фоновых изображений. Оставшееся незаполненное место распределяется в равномерные отступы между экземплярами изображения.
  • round — повторять так, чтобы в области поместилось целое число фоновых изображений, при необходимости их растягивая или сжимая, чтобы заполнить оставшееся пространство.

Свойство может принимать в качестве значения два аргумента. В этом случае, первое значение задаст режим повторения по горизонтали, а второе — по вертикали.

Примеры:

background-repeat: space no-repeat;
background-repeat: repeat round;

По возможности, вместо одного большого изображения, стоит использовать одно маленькое со значением свойства repeat. Это может быть применимо, когда изображение состоит из повторяющихся фрагментов. Это обеспечит более быструю загрузку страницы из-за меньшего веса изображения.

Эффекты с повторяющимся фоном

Повторяющийся фон часто используется для создания интересных декоративных эффектов. Например, стежков, зазубренных краёв, градиентов и теней.

Да, сейчас уже градиенты и тени можно делать с помощью CSS. Но иногда, когда тень или градиент достаточно сложные, проще использовать полупрозрачный PNG.

Самое главное при создании таких эффектов — выбрать картинку с нужным периодом. Она может быть очень маленькой и даст существенную экономию веса страницы.
