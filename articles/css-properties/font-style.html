Свойство font-style

Свойство font-style задаёт стиль начертания шрифта.

Значения:

  • normal — (по умолчанию) обычное начертание.
  • italic — курсивное начертание.
  • oblique — наклонное начертание.

При значении italic браузер попытается найти в заданном шрифте отдельное курсивное начертание символов. Если это не удастся, браузер сымитирует курсив, сделав текст наклонным, как при значении oblique.
