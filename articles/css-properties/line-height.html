Свойство line-height

Свойство line-height задаёт высоту строки.

Значения:

  • normal — (по умолчанию) соотвествует 100–120%.
  • Значение в относительных или абсолютных единицах измерения.
  • Множитель.

Относительные значения вычисляются от размера шрифта элемента.

Пример:

font-size: 10px;
line-height: 2; /* Вычисленное значение: 10px * 2 = 20px */
