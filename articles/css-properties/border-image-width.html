Свойство border-image-width.

Область, в которой будет отображаться фоновое изображение рамки, по умолчанию равна ширине рамки.

Свойство border-image-width позволяет управлять шириной области рамки-изображения, масштабировать её. Ширину самой рамки это свойство не меняет.

Если значение этого свойства больше border-width, фоновое изображение рамки заползёт под содержимое, даже если не задано свойство fill.

Свойство принимает от одного до четырёх аргументов в любых единицах длины, разделённых между собой пробелом:

  • 1 — одинаковая ширина всех сторон.
  • 2 — первое значение устанавливает ширину сверху и снизу, второе — слева и справа.
  • 3 — первое значение устанавливает ширину сверху, второе — слева и справа, а третье — снизу.
  • 4 — поочередно устанавливаются ширина сверху, справа, снизу и слева.

Примеры:

border-image-width: 10px 20px 30px 40px;
border-image-width: 10px 50px;

Значение по умолчанию — auto, при котором ширина зависит от значения border-image-slice.
