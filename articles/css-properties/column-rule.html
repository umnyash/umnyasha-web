Свойство column-rule

Свойство column-rule задаёт стили разделительной линии между колонками.

Составное свойство:

  • column-rule-width — ширина.
  • column-rule-style — стиль.
  • column-rule-color — цвет.

Значения:

  • Параметры разделительной линии, перечисленные через пробел.

Пример:

column-rule: 1px solid #cccccc;
