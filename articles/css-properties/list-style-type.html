Свойство list-style-type.

Свойство list-style-type задаёт вид номеров/маркеров в упорядоченных и неупорядоченных списках.

  • none — удаление номеров/маркеров. При этом занимаемое ими место не резервируется.

Значения неупорядоченные:

  • disc — (по умолчанию) закрашенный круг.
  • circle — незакрашенный круг.
  • square — квадрат.

Значения упорядоченные:

  • decimal — (по умолчанию) обычные арабские цифры.
  • decimal-leading-zero — арабские цифры, до десятков будут писаться с нулём в начале.
  • lower-greek — строчные буквы греческого алфавита.
  • lower-latin — строчные буквы латинского алфавита.
  • lower-roman — строчные римские цифры.
  • upper-latin — прописные буквы латинского алфавита.
  • upper-roman — прописные римские цифры.
