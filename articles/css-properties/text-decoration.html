Свойство text-decoration

Свойство text-decoration задаёт стили декоративной линии.

Составное свойство:

  • text-decoration-line — тип.
  • text-decoration-style — вид.
  • text-decoration-color — цвет.

Значения:

  • Перечисленные через пробел в любом порядке значения стилей.
  • none — удаление линии.

Пример:

text-decoration: underline wavy coral;

Можно задать несколько линий, если их стиль и цвет одинаковые:

text-decoration: overline line-through underline solid green;
