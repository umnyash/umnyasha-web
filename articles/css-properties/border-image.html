Свойство border-image

Свойство border-image задаёт фоновое изображение рамки.

Составное свойство:

  1. border-image-source — изображение.
  2. border-image-slice — нарезка.
  3. border-image-width — ширина.
  4. border-image-outset — отступ.
  5. border-image-repeat — режим заполнения.

Значения:

  • Параметры фонового изображения рамки, перечисленные через пробел.

Сначала

Примеры:

/* source slice */
border-image: linear-gradient(red, blue) 27;

/* source slice repeat */
border-image: url("/images/border.png") 27 space;

/* source slice width */
border-image: linear-gradient(red, blue) 27 / 35px;

/* source slice width outset repeat */
border-image: url("/images/border.png") 27 23 / 50px 30px / 1rem round space;
