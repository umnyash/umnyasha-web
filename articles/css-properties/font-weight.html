Свойство font-weight.

Свойство font-weight задаёт толщину шрифта.

Значения:

  • Ключевое число.
  • Ключевое слово.

Ключевые числа:

  1. 100 — Thin (Hairline).
  2. 200 — Extra Light (Ultra Light).
  3. 300 — Light.
  4. 400 — Normal (Regular) — обычный шрифт, значение по умолчанию.
  5. 500 — Medium.
  6. 600 — Semi Bold (Demi Bold).
  7. 700 — Bold — жирный шрифт.
  8. 800 — Extra Bold (Ultra Bold).
  9. 900 — Black (Heavy).
  10. 950 — Extra Black (Ultra Black).

Ключевые слова для абсолютных значений:

  • normal — (по умолчанию) 400.
  • bold — 700.

Поскольку ключевые слова есть не для всех значений, то для единообразия лучше их не использовать.

Ключевые слова для относительных значений:

  • lighter.
  • bolder.
