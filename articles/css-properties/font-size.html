Свойство font-size

Свойство font-size задаёт размер шрифта.

Значения:

  • Значение в относительных или абсолютных единицах измерения.
  • Ключевое слово.

Ключевые слова для абсолютных значений (по возрастанию):

  1. xx-small.
  2. x-small.
  3. small.
  4. medium.
  5. large.
  6. x-large.
  7. xx-large.

Ключевые слова для относительных значений:

  • larger.
  • smaller.
