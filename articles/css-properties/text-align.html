Свойство text-align.

Свойство text-align задаёт выравнивание текста и других инлайновых элементов по горизонтали.

Значения:

  • left — (по умолчанию) выравнивание по левому краю блока.
  • right — выравнивание по правому краю блока.
  • center — выравнивание по центру.
  • justify — выравнивание по ширине блока, при этом слова в строке будут размещены так, чтобы равномерно занять всё пространство строки.
  • start — выравнивание к началу блока.
  • end — выравнивание к концу блока.
