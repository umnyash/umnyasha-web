Свойство border.

Свойство border создаёт рамку элемента.

Составное свойство:

  • border-width — ширина.
  • border-style — вид.
  • border-color — цвет.

Значения:

  • Параметры рамки, перечисленные через пробел в любом порядке.
  • none — удаление рамки.

Пример:

border: 2px solid grey;
