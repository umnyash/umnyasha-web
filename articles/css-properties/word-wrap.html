Свойство word-wrap.

Свойство word-wrap управляет переносом слов, которые не помещаются по ширине.

Значения:

  • normal — (по умолчанию) непомещающаяся часть слова отображается за пределами элемента.
  • break-word — непомещающаяся часть слова переносится на новую строку.

Это свойство не описано в спецификации. Браузеры реализовали его на своё усмотрение, когда ещё не было overflow-wrap.
