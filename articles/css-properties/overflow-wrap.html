Свойство word-wrap.

Свойство overflow-wrap управляет переносом слов, которые не помещаются по ширине.

Значения:

  • normal — (по умолчанию) непомещающаяся часть слова отображается за пределами элемента.
  • break-word — непомещающаяся часть слова переносится на новую строку.
  • anywhere — непомещающаяся часть слова также переносится на новую строку.

Различие между break-word и anywhere в том, как будет вычисляться ширина элемента с текстом при значении min-content. При значении break-word слова будут оставаться целыми, а при anywhere будут разделены.
