Свойство cursor.

Свойство cursor задаёт тип курсора.

Значения:

  • auto — (по умолчанию) браузер самостоятельно определяет значение курсора в зависимости от свойств элемента, на который наведена мышь.
  • default — основной курсор платформы, обычная стрелочка.
  • pointer — указатель, обозначающий ссылку. Обычно указующая рука.
  • none — курсор не отображается.

Значение pointer можно использовать при декорировании кнопок, чтобы пользователю было очевидно, что перед ним именно активный элемент, на который можно нажать.
