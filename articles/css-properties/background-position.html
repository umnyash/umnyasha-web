Свойство background-posititon

Свойство background-position задаёт позицию фонового изображения.

В качестве значения принимает два аргумента: позицию по горизонтали и позицию по вертикали.

Расположение по горизонтали можно задать ключевыми словами left, center и right, а расположение по вертикали — словами top, center и bottom.

Помимо ключевых слов можно использовать значения в пикселях и процентах, в том числе значения отрицательные.

Если аргумент один, то он принимается за позицию по горизонтали, либо за позицию, соответствующую ключевому слову, если оно используется. Второе значение принимается за center.

Значение по умолчанию: 0 0.

Примеры:

/* Картинка по центру. */
background-position: center;
background-position: 50% 50%;

/* В левом верхнем углу. */
background-position: 0 0;

/* В левом нижнем углу. */
background-position: left bottom;
background-position: 0 100%;

/* На расстоянии 20px от левого края и на 40px от верхнего. */
background-position: 20px 40px;

Позиция от разных сторон

Позицию фонового изображения можно задавать относительно любого угла блока, а не только от левого верхнего.

Чтобы указать от какой стороны отсчитывать расположение фона, нужно перед значением координат указать ключевые слова: top, right, bottom или left.

Примеры:

/* Слева 10px, сверху 50px. */
background-position: 10px 50px;
background-position: left 10px top 50px;

/* Справа 30px, снизу 60px. */
background-position: right 30px bottom 60px;

/* Слева 50px, снизу 10px. */
background-position: left 50px bottom 10px;

/* Справа 40px, сверху 30px. */
background-position: right 40px top 30px;
