Свойство list-style-image.

Свойство list-style-image задаёт картинку, которая будет отображаться вместо маркеров.

Пример:

list-style-image: url("rocket.svg");

Если картинка не загрузится, то отобразится заданный маркер.

Если вид списка требует иконок, лучше вставлять их как фон для элементов списка или их псевдоэлементов.
