Свойство outline

Свойство outline создаёт внешнюю рамку элемента.

Составное свойство:

  • outline-width — ширина.
  • outline-style — вид.
  • outline-color — цвет.

Значения:

  • Параметры рамки, перечисленные через пробел в любом порядке.
  • none — удаление рамки.

Пример:

outline: 20px dashed green;
