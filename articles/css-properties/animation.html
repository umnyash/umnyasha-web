Свойство animation

Свойство animation задаёт параметры анимации.

Составное свойство:

  • animation-name — название анимации.
  • animation-duration — длительность.
  • animation-timing-function — темп.
  • animation-delay — задержка.
  • animation-iteration-count — количество итераций.
  • animation-direction — направление.
  • animation-fill-mode — режим применения стилей.
  • animation-play-state — состояние.

Значения:

  • Перечисленные через пробел параметры для одной или нескольких анимаций, перечисленных через запятую.

Пример:

animation: name-1 1s ease-in 1s 2 reverse both;
animation: name-1 1s ease-in 1s 2 reverse both, name-2 4s ease-out 4s 3 alternate forwards;
