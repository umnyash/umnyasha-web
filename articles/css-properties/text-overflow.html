Свойство text-overflow

Свойство text-overflow позволяет определить, как будет выглядеть текст, который не помещается полностью.

Значения:

  • clip — (по умолчанию) текст просто обрезается.
  • ellipsis — текст обрезается и к концу строки добавляется многоточие.

Для удобства пользователя элементу можно добавить атрибут title с полным текстом.

Требования для применения:

  • Определено максимальное количество строк.
  • Значение свойства overflow установлено как auto, scroll или hidden.

Пример однострочного текста

Чтобы сделать текст однострочным можно использовать свойство white-space со значением nowrap:

p {
  width: 200px;
  padding: 4px 8px;
  border-radius: 4px;
  white-space: nowrap;
  text-overflow: ellipsis;
  background: lightblue;
  overflow: hidden;
}

Пример многострочного текста

Чтобы задать максимальное количество строк понадобятся display: -webkit-box, -webkit-box-orient: vertical и -webkit-line-clamp.

p {
  display: -webkit-box;
  width: 200px;
  padding: 4px 8px;
  border-radius: 4px;
  -webkit-line-clamp: 3; /* количество строк */
  -webkit-box-orient: vertical;
  text-overflow: ellipsis;
  background: lightblue;
  overflow: hidden;
}
