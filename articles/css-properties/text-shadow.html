Свойство text-shadow.

Свойство text-shadow задаёт тень тексту.

Особенности:

  • Применяется к тексту.
  • Форма тени повторяет форму текстовых символов.
  • Можно задать смещение, размытие и цвет тени.
  • Нельзя задать растяжение.
  • Можно создавать множественные тени.

Значения:

  • Перечисленные через пробел параметры для одной или нескольких теней, перечисленных через запятую.
  • none — (по умолчанию) отсутствие тени.

Пример:

text-shadow: 0 0 5px #333333;
  • Первый параметр задаёт смещение тени по горизонтали. Положительное значение сдвигает тень вправо, отрицательное — влево.
  • Второй параметр задаёт смещение тени по вертикали. Положительное значение сдвигает тень вниз, отрицательное — вверх.
  • Третий параметр задаёт радиус размытия тени. Чем больше значение параметра, тем сильнее тень размывается и становится светлее. Значение по умолачнию — 0.
  • Последний параметр задаёт цвет тени. По умолчанию цвет тени такой же, как цвет текста.

Обязательны только параметры смещения.

У текста может быть несколько теней. При этом тени распределятся по правилу: первая тень в списке — самая верхняя, последняя — самая нижняя.

Пример:

text-shadow: 1px 1px 1px #111111, 2px 2px 2px #222222;

С помощью множественных теней можно добиваться интересных эффектов.

Эффект вдавленного текста.

Тексту можно добавить эффект «вдавленности» с помощью двух однотонных теней: более тёмная смещается немного вверх и влево, а более светлая – вниз и вправо.

p {
  padding: 20px;
  font: 400 60px/1 "Arial", sans-serif;
  text-transform: uppercase;
  color: #2b2328;
  text-shadow: -1px -1px 1px #1e181c, 2px 2px 1px #52424d;
  background: #42353e;
}

Декоративная ретро-тень.

Эффект обводки создаётся из двух резких теней, смещённых в одну сторону. Нижняя тень смещена чуть сильнее и её цвет отличается от цвета фона, а верхняя тень смещена слабее и цвет её совпадает с цветом фона.

Такой эффект хорошо подходит для заголовков с винтажным стилем.

p {
  padding: 20px;
  font: 400 60px/1 "Arial", sans-serif;
  text-transform: uppercase;
  color: #444444;
  background: #e5d4c0;
  text-shadow: 2px 2px #e5d4c0, 4px 4px rgba(0, 0, 0, 0.2);
}
