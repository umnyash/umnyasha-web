Свойство font

Свойство font задаёт основые параметры шрифта.

Составное свойство:

  • font-family — семейство.
  • font-size — размер.
  • line-height — высота.
  • font-weight — толщина.
  • font-style — стиль.
  • font-variant — вариант отображения символов.

Значения:

  • Параметры шрифта, перечисленные через пробел.

Обязательными параметрами являются значения размера и семейства шрифта:

font: 20px "PT Sans", "Arial", sans-serif;

Высота строки всегда указывается сразу после размера шрифта через слэш (/). Значение высоты строки auto недопустимо использовать в этом свойстве. Значение размера шрифта или пара размер шрифта/высота строки всегда указываются непосредственно перед семейством шрифта:

font: 20px "Georgia", serif;
font: 20px/30px "Georgia", serif;

Остальные параметры (толщина, стиль, вариант) могут указываться в произвольном порядке до размера шрифта:

font: bold 20px "Verdana", sans-serif;
font: small-caps 700 italic 20px/30px "Georgia", serif;
font: italic small-caps 700 20px/30px "Georgia", serif;
