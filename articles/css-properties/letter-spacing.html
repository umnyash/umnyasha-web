Свойство letter-spacing.

Свойство letter-spacing увеличивает межсимвольное расстояние на заданную величину.

Значения:

  • Значение в относительных или абсолютных единицах измерения.
  • normal — (по умолчанию) соотвествует 0.

Значения в % не поддерживаются, но их можно задать с помощью em:

letter-spacing: 0.05em; /* равнозначно 5% */
