Свойство writing-mode.

Свойство writing-mode задаёт режим печати текста.

Значения:

  • horizontal-tb — (по умолчанию) текст печатается слева направо, порядок строк: сверху вниз.
  • vertical-rl — текст печатается сверху вниз, порядок строк: справа налево.
  • vertical-lr — текст печатается сверху вниз, порядок строк: слева направо.
