Свойство word-break.

Свойство word-break управляет переносом слов, которые не помещаются по ширине.

Значения:

  • normal — (по умолчанию) непомещающаяся часть слова отображается за пределами элемента.
  • break-word — непомещающаяся часть слова переносится на новую строку.
  • break-all — непомещающаяся часть слова переносится на новую строку, даже если слово могло бы быть перенесено целиком.
