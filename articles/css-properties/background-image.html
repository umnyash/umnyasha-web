Свойство background-image.

Свойство background-image задаёт фоновое изображение.

Значения:

  • Адрес изображения внутри url("...").

Пример:

background-image: url("img/sky.jpg");

Когда фоновое изображение больше блока, оно обрезается. Управлять тем, какую часть изображения будет видно, можно с помощью свойства background-position. Иногда для этого удобно использовать относительные значения, а иногда — абсолютные.
