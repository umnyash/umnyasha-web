Свойство background-size.

Свойство background-size задаёт размеры фонового изображения.

Принимает в качестве значения два аргумента: первый — это ширина, второй (необязательный) — высота. Если высота не указывается, то определяется автоматически в зависимости от ширины и исходных пропорций.

По умолчанию браузер не меняет размеры фонового изображения и оба аргумента равны auto.

Примеры:

background-size: auto auto; /* Исходные ширина и высота. */
background-size: 100px; /* Ширина 100px, высота пропорциональная. */
background-size: 100px 200px; /* Ширина 100px, высота 200px. */
background-size: 100%; /* Ширина равна ширине родительского блока, высота пропорциональная. */
background-size: 100% 100%; /* Ширина и высота равны ширине и высоте родительского блока. */

Ключевые слова для значений: contain и cover.

При значении contain:

  • Пропорции изображения сохраняются.
  • Изображению задаются максимально возможные размеры, при которых оно по ширине и по высоте полностью помещается в границы блока.
  • Изображение может не закрывать всю фоновую область блока, если пропорции изображения и блока разные.

При значении cover:

  • Пропорции изображения сохраняются.
  • Изображению задаются минимально возможные размеры, при которых оно покроет всю фоновую область блока.
  • Если пропорции изображения и блока разные, то часть изображения обрезается.
