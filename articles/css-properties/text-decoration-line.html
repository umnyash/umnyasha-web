Свойство text-decoration-line.

Свойство text-decoration-line задаёт тип декоративной линии.

Значения:

  • underline — подчёркивание.
  • line-through — зачёркивание.
  • overline — надчёркивание.
  • none — (по умолчанию) отсутствие линии.

Одному элементу можно задать несколько линий, перечислив их через пробел:

text-decoration-line: underline overline line-through;
