Псевдокласс not.

Псевдокласс not(селектор) является отрицающим селектором. С его помощью можно выбрать элементы, которые НЕ соответствуют указанному селектору.

В качестве отрицаемого селектора могут указываться псевдоклассы, теги, идентификаторы, классы и селекторы атрибутов.

Пример:

li:not(:last-child) { ... }

Этот селектор выберет все теги <li></li>, НЕ являющиеся последними в их родителе.

Нельзя использовать двойной псевдокласс not, то есть конструкция li:not(:not(...)) не сработает.

Также в комбинации с not не применяются:

  • Объединение селекторов: например, li:not(.heart.jack) — неккоректный селектор.
  • Псевдоэлементы: li:not(::after) — неправильная запись.
  • Селекторы-потомки, групповые селекторы или комбинации: например, нельзя писать li:not(a span) или li:not(a + span).

Псевдокласс not имеет нулевую специфичность.

Комбинации с not.

Отрицающий селектор not, как и любые другие селекторы, можно комбинировать с другими.

Пример:

li:not(:first-child):not(:last-child) { ... }

Выберет все теги <li></li>, которые НЕ являются первыми и последними в их родителе.
