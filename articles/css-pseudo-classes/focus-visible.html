Псевдокласс focus-visible.

Элемент попадает в фокус при клике на него, а также при последовательном нажатии кнопки Tab. С помощью псевдокласса focus описываются стили для элемента, который оказался в фокусе любым способом.

Псевдокласс focus-visible задаёт стили элемента, которые отображаются только когда элемент оказался в фокусе при последовательной нажатии кнопки Tab. Когда элемент оказывается в фокусе при клике, эти стили не отображаются. Это соответствует собственному поведению фокуса для таких элементов управления, как <button></button>.
