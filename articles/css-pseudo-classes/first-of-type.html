Псевдокласс first-of-type.

Псевдокласс first-of-type выбирает первые теги разных типов среди дочерних.

Пример:

<!-- HTML -->
<html>
  <head></head>
  <body>
    <div></div>
    <div></div>
    <p></p>
    <p></p>
    <p></p>
  </body>
</html>

/* CSS */
:first-of-type {
  ...
}

Будут выбраны теги <html></html>, <body></body>, первый <div></div> и первый <p></p>.

Обычно используется в комбинации с другими селекторами.
