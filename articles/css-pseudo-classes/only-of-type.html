Псевдокласс only-of-type.

Псевдокласс only-of-type выбирает элемент, если он является единственным дочерним тегом своего типа.

Пример:

p:only-of-type { ... }

Тег <p></p> будет выбран, если он является единственным параграфом в своём родительском теге.

Селектор :only-of-type эквивалентен комбинации :first-of-type:last-of-type.
