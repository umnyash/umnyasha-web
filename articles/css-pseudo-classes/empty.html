Псевдокласс empty.

Псевдокласс empty выбирает пустые теги: те у которых нет ни дочерних элементов, ни текста внутри.

Пример:

div:empty { ... }

В данном случае будут выбраны все пустые <div></div>.

Учтите, что даже переход на новую строку считается текстовым узлом, помните об этом в процессе проектирования структуры страницы.
