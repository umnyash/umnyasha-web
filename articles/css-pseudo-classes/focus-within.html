Псевдокласс focus-within.

Псевдокласс focus-within выбирает элемент, если он сам или его потомок находятся в фокусе.

В примере ниже фон для обёртки меняет фоновый цвет на жёлтый, когда ссылка оказывается в фокусе:

<!-- HTML -->
<div class="wrapper">
  <a href="#">Ссылка</a>
</div>

/* CSS */
.wrapper:focus-within {
  background-color: yellow;
}
