Псевдокласс last-of-type.

Псевдокласс last-of-type выбирает последние теги разных типов среди дочерних.

Пример:

<!-- HTML -->
<html>
  <head></head>
  <body>
    <div></div>
    <div></div>
    <p></p>
    <p></p>
    <p></p>
  </body>
</html>

/* CSS */
:last-of-type {
  ...
}

Будут выбраны теги <html></html>, <body></body>, второй <div></div> и третий <p></p>.

Обычно используется в комбинации с другими селекторами.
