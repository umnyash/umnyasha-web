Псевдокласс only-child.

Псевдокласс only-child выбирает элемент, если он является единственным в своём родительском теге.

Например, этот тег <li></li>, будет выбран, если он единственный в списке:

li:only-child { ... }

Селектор :only-child эквивалентен комбинации :first-child:last-child.
