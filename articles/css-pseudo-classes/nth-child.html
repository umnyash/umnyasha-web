Псевдокласс nth-child.

Псевдокласс nth-child выбирает элемент по порядковому номеру, который отсчитывается с начала.

В качестве аргумента принимает положительное число, ключевое слово или выражение со счётчиком.

Ключевые слова:

  • even — все чётные номера.
  • odd — все нечётные номера.

Примеры:

li:nth-child(1) { ... }   /* 1-й тег li в списке. */
li:nth-child(4) { ... }   /* 4-й тег li в списке. */
li:nth-child(odd) { ... } /* 1-й, 3-й, 5-й и т.д. */

Использование счётчика.

Счётчик n принимает целые значения, начиная с нуля: 0, 1, 2, 3, 4 и так далее.

Выбор каждого n-ного элемента.

Рассмотрим пример вычисления номеров для выбора каждого второго элемента:

li:nth-child(2n) { ... }

Вычисление номеров элементов:

  • n = 0 ➝ 2 * 0 ➝ 0 (нет элементов)
  • n = 1 ➝ 2 * 1 ➝ 2 (2-й элемент)
  • n = 2 ➝ 2 * 2 ➝ 4 (4-й элемент)
  • n = 3 ➝ 2 * 3 ➝ 6 (6-й элемент)
  • ...

Таким образом, выражение 2n выбирает каждый второй (чётный) элемент, так же как ключевое слово even.

Используя разные стили для чётных и нечётных элементов, можно создавать интересные визуальные эффекты. Именно таким образом, с помощью чистого CSS, делают полосатые таблицы.

Выбор каждого n-ного элемента со сдвигом.

Выбор нечётных элементов.

Посчитаем номера для 2n + 1:

  • n = 0 ➝ 2 * 0 + 1 ➝ 1 (1-й элемент)
  • n = 1 ➝ 2 * 1 + 1 ➝ 3 (3-й элемент)
  • n = 2 ➝ 2 * 2 + 1 ➝ 5 (5-й элемент)
  • n = 3 ➝ 2 * 3 + 1 ➝ 7 (7-й элемент)
  • ...

Выражение 2n + 1 также как и 2n выбирает каждый второй элемент, но начинает не со второго элемента, а с первого. Это выражение равнозначно ключевому слову odd.

Выбор каждого третьего элемента, начиная с первого.

Рассмотрим ещё один пример:

li:nth-child(3n - 2) { ... }

Вычисление номеров элементов:

  • n = 0 ➝ 3 * 0 - 2 ➝ -2 (нет элементов)
  • n = 1 ➝ 3 * 1 - 2 ➝ 1 (1-й элемент)
  • n = 2 ➝ 3 * 2 - 2 ➝ 4 (4-й элемент)
  • n = 3 ➝ 3 * 3 - 2 ➝ 7 (7-й элемент)
  • ...

Выражение 3n - 2 также как и 3n выбирает каждый третий элемент, но начинает не с третьего элемента, а с первого.

Выбор первых нескольких элементов.

Например, -n + 2 выбирает только первые два элемента:

  • n = 0 ➝ 0 + 2 ➝ 2 (2-й элемент)
  • n = 1 ➝ -1 + 2 ➝ 1 (1-й элемент)
  • n = 2 ➝ -2 + 2 ➝ 0 (нет элементов)
  • n = 3 ➝ -3 + 2 ➝ -1 (нет элементов)
  • ...

Выбор всех элементов, кроме начальных.

Например, n + 8 выбирает все элементы, начиная с восьмого:

  • n = 0 ➝ 0 + 8 ➝ 8 (8-й элемент)
  • n = 1 ➝ 1 + 8 ➝ 9 (9-й элемент)
  • n = 2 ➝ 2 + 8 ➝ 10 (10-й элемент)
  • n = 3 ➝ 3 + 8 ➝ 11 (11-й элемент)
  • ...

Комбинирование выражений.

Существует возможность задавать множество с помощью комбинирования псевдоклассов:

Рассмотрим пример выбора элементов в диапазоне 10–15:

li:nth-child(n + 10):nth-child(-n + 15) { ... }
  1. n + 10 выберет все элементы, начиная с 10-го.
  2. Из них выражению -n + 15, которое выбирает первые 15 элементов, остаётся выбрать только элементы с 10-го по 15-й.

Можно комбинировать более двух выражений.

Пример:

li:nth-child(n+4):nth-child(-n+18):nth-child(odd) { ... }

Этот селектор выберет нечётные элементы среди элементов с 4-го по 18-й включительно.
