Псевдокласс focus.

Псевдокласс focus выбирает элемент, который находится в фокус.

Например, текстовое поле, в которое установлен курсор, находится в фокусе.

Пример:

input:focus {
  /* Стили для поля в фокусе. */
}

В фокусе могут быть не только текстовые поля. Если вы переключаетесь между элементами веб-страницы с помощью клавиши Tab, то в фокус будут попадать ссылки и кнопки.
