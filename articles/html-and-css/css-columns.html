CSS-колонки

CSS-колонки — замечательная возможность, позволяющая разбить блок с текстом на несколько колонок. Обычно очень длинные строки сложно читать. Чтобы сэкономить место на экране, но при этом сделать текст читабельным, можно разбить один сплошной блок текста на несколько колонок, как это делается в газетах.

Неважно, какие элементы находятся внутри блока, который вы превращаете в многоколоночный контейнер, все элементы остаются в обычном потоке, но разбиваются на колонки.

  • column-count — количество колонок.
  • column-width — минимальная ширина колонок.
  • columns — составное свойство (count width).
  • column-gap — промежуток между колонками.
  • column-rule — разделительная линия между колонками.
  • column-span — охват колонок.
