Веб-безопасные шрифты

Веб-безопасные шрифты — это те, которые установлены на компьютерах максимального количества пользователей.

Статистика веб-безопасных шрифтов на разных операционных системах.

При использовании нестандартного шрифта в качестве основного, важно указать альтернативный веб-безопасный шрифт, которым воспользуется браузер в случае, если основной не загрузится. Его нужно подобрать максимально похожим на основной шрифт.

Важно, чтобы запасные шрифты поддерживали язык основного шрифта, например, кириллицу, если текст на сайте написан по-русски.

font-family: "Roboto", "Arial", sans-serif;
