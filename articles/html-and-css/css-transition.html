CSS-переходы

CSS transition (CSS-переходы) позволяют плавно менять исходное значение CSS-свойства на новое вместо мгновенного переключения.

  • transition-property — свойства, к которым будет применён переход.
  • transition-duration — длительность.
  • transition-timing-function — темп.
  • transition-delay — задержка перед началом перехода.
  • transition — составное свойство (property duration timing-function delay).

Чтобы обозначить плавный переход, достаточно задать его длительность:

transition-duration: 300ms;

Для разных свойств можно задавать разные параметры перехода. Для этого параметры нужно перечислить через запятую в том же порядке, что и названия свойства:

transition-property: width, height;
transition-duration: 1s, 5s; /* Ширина меняется за 1 секунду, высота — за 5. */
