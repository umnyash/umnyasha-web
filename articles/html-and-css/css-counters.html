CSS-счётчики.

Стандартные числовые или буквенные маркеры сложно стилизовать. Поэтому их часто скрывают с помощью list-style: none и затем воссоздают с помощью CSS-счётчиков.

Свойство counter-reset, инициализация счётчика.

Свойство counter-reset задаётся родителю тех элементов, которые нужно считать. Оно создаёт счётчик и устанавливает его начальное значение.

.list {
  counter-reset: years 2000;
}

Если начальное значение не задано, то оно будет равняться нулю.

Свойство counter-increment, подключение счётчика.

Свойство counter-increment принимает в качестве значения название счётчика и задаётся тем элементам, которые нужно считать.

.list li {
  counter-increment: years;
}

В данном примере, в значении счётчика years будет храниться только число дочерних элементов <li></li>.

Элементы <li></li> вложенного списка (при наличии) будут подсчитываться отдельно счётчиком с таким же названием.

Вывод значения счётчика.

Значение счётчика можно вывести с помощью псевдоклассов ::before/::after, свойства content и функций counter() и counters().

Содержимое content может собираться из нескольких частей, например:

.list li::before {
  content: "#" counter(years);
}

Получится — #2001, #2002 и так далее.

Чтобы вложенные элементы наследовали значение счётчика родителя, нужно воспользоваться функцией counters(). Она принимает два параметра: название счётчика и строку разделитель, которая будет появляться между значениями вложенных счётчиков.

Пример:

<!-- HTML -->
<ol>
  <li>HTML</li>
  <li>Тип документа</li>
  <li>Основные теги HTML
    <ol>
      <li>Тег html</li>
      <li>Тег head</li>
      <li>Тег body</li>
    </ol>
  </li>
  <li>Крупные смысловые блоки</li>
</ol>

/* CSS */
ol {
  list-style: none;
  counter-reset: chapters;
}
ol li {
  counter-increment: chapters;
}
ol li::before {
  content: counters(chapters, ".") ". ";
}

Если счётчик инициализирован не в родителе элементов, то дочерние элементы и другие потомки будут считаться вместе:

<!-- HTML -->
<nav>
  <ol>
    <li>HTML</li>
    <li>Тип документа</li>
    <li>Основные теги HTML
      <ol>
        <li>Тег html</li>
        <li>Тег head</li>
        <li>Тег body</li>
      </ol>
    </li>
    <li>Крупные смысловые блоки</li>
  </ol>
</nav>

/* CSS */
nav {
  counter-reset: chapters;
}
ol {
  list-style: none;
}
ol li {
  counter-increment: chapters;
}
ol li::before {
  content: counters(chapters, ".") ". ";
}
