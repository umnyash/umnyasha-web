Unicode-символы.

В стандарте кодирования Unicode представлены знаки почти всех письменных языков, а также разные специальные символы.

Многие их них непросто набрать на клавиатуре, а некоторые зарезервированы в HTML, то есть браузер считает их HTML кодом. Например, любой текст после знака < («меньше») браузер будет пытаться интерпретировать как тег и на странице не отобразит.

В разметке возможно написать любые символы Unicode в числовом обозначении (HTML-коды), или используя символы-мнемоники.

Числовые коды запомнить сложно, а мнемоники — это буквенные коды, имеющие человекопонятное значение. Каждая мнемоника начинается с амперсанда (&) и заканчивается точкой с запятой (;).

Примеры Unicode-символов:

&#169;    &copy;    ©
&#945;    &alpha;   α
&#182;    &para;    ¶
&#91;     &lbrack;  [
&#93;     &rbrack;  ]

Например, знак < на страницу можно вставить мнемоникой &lt; (less than), а знак > мнемоникой &gt; (greater than).

&lt;ul&gt;&lt;/ul&gt;

Юникод-символы могут быть использованы в стилях с помощью своих CSS-кодов, например, в качестве значений свойства content:

.rub::after {
  content: "\20BD"; /* использован символ рубля */
}
