CSS-анимации.

CSS animation (CSS-анимации) имеют больше возможностей и помогают сделать то, что не могут CSS-переходы. В том числе создавать анимации с количеством шагов больше двух или запускаться сразу же при загрузке страницы и повторяться любое количество раз без какого-либо действия пользователя.

Создание анимации.

Анимация создаётся с помощью правила @keyframes, в котором нужно задать название и описать изменяемые CSS-свойства в ключевых кадрах.

Требования к названию анимации:

  • Можно использовать только латинские буквы, цифры, знаки дефиса (-) и нижнего подчёркивания (_).
  • Нельзя ставить два дефиса подряд в начале имени.
  • Первый символ, или первый после дефиса, должен быть буквой.
  • Нельзя использовать специальные слова вроде inherit, none, initial, unset и т.п.

Ключевые кадры задаются в процентах от времени анимации. Значения 0% и 100% могут быть заменены ключевыми словами from и to.

Пример:

@keyframes coloring { /* Эта анимация называется "coloring". */
  0% { color: red; }
  33% { color: yellow; }
  66% { color: green; }
  100% { color: blue; }
}

Если начальный или конечный кадр не задан, то он автоматически создаётся, принимая исходное стилевое состояние элемента.

Задавать ключевые кадры можно в любом порядке. Это может быть удобно, чтобы сгруппировать одинаковые кадры. В таком случае их можно перечислить через запятую:

@keyframes example {
  0%, 100% { ... }
  10%, 30%, 50%, 70%, 90% { ... }
  20%, 40%, 60%, 80% { ... }
}

Применение анимации.

  • animation-name — название.
  • animation-duration — длительность.
  • animation-timing-function — темп.
  • animation-delay — задержка.
  • animation-iteration-count — количество итераций.
  • animation-direction — направление.
  • animation-fill-mode — режим применения стилей.
  • animation-play-state — состояние.
  • animation — составное свойство (name duration timing-function delay iteration-count direction fill-mode play-state).

Чтобы применить анимацию к элементу, достаточно задать её имя и длительность.

По умолчанию, после завершения анимации элемент возвращается в исходное состояние.

Одному элементу можно задать несколько анимаций.

Для разных анимаций можно задавать разные параметры. Для этого параметры нужно перечислить через запятую в том же порядке, что и названия свойства:

animation-name: move, stretch;
animation-duration: 5s, 3s;

Если разными анимациями одновременно должно изменяться одно и то же свойство, то оно будет изменяться в соответствии с анимацией, заданной позже других.
