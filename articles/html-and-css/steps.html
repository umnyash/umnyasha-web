Функция steps.

При использовании функции steps переходы и анимации будут происходить пошагово.

steps(number, start/end)

Длительность перехода или анимации делится на равные отрезки времени в соответствии с количеством шагов (number).

Второй параметр влияет на то, когда в пределах каждого отрезка времени будет происходить изменение CSS-свойств: в начале или в конце. Значение end — по умолчанию.

При значении start анимация фактически будет завершаться чуть раньше.

Псевдонимы функций:

  • step-start — равнозначен steps(1, start). Стилевые свойства сразу принимают конечные значения.
  • step-end — равнозначен steps(1, end). Стилевые свойства находятся в начальном значении заданное время, затем сразу принимают конечные значения.
