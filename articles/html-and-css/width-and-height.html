Ширина и высота элементов.

Ширина и высота элементов задаются с помощью свойств width и height.

Пример:

width: 200px;
height: 100px;

При этом задаются не общие ширина и высота блока, а только ширина и высота его содержания. Общие размеры затем складываются из трёх компонентов: ширины и высоты содержания, внутренних отступов и ширины рамок.

Изменить алгоритм расчёта ширины и высоты можно с помощью свойства box-sizing.

Вернуть значения по умолчанию можно с помощью значения auto:

width: auto;
height: auto;

Максимальные и минимальные размеры задаются следующими свойствами:

  • min-width — минимальная ширина.
  • max-width — максимальная ширина.
  • min-height — минимальная высота.
  • max-height — максимальная высота.
