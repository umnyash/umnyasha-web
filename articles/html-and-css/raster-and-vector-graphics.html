Растровая и векторная графика

Растровая графика.

Растровое изображение, как мозаика, складывается из множества маленьких ячеек — пикселей, где каждый пиксель содержит информацию о цвете.

Растровая графика удобна для создания качественных фотореалистичных изображений, рисунков и фотографий.

Преимущества:

  • Возможность создать изображение любой сложности — с огромным количеством деталей и широкой цветовой гаммой.

Недостатки:

  • Большой объём занимаемой памяти: чем больше ширина и высота растрового изображения, тем больше в нём пикселей и, соответственно, тем больше места нужно для хранения/передачи такого изображения.
  • При изменении размера оригинального растрового изображения неизбежно произойдёт потеря качества (в результате процесса интерполяции).

Наиболее распространённые растровые форматы: JPEG, PNG.

Векторная графика.

Векторные изображения состоят из множества опорных точек и соединяющих их кривых, описанных с помощью математических формул.

Векторная графика используется для иллюстраций, иконок, логотипов и технических чертежей.

Преимущества:

  • Простые векторные изображения занимают малый объём памяти, так как содержат малое количество информации.
  • Векторные изображения масштабируются без потерь качества.

Недостатки:

  • Чтобы отобразить векторное изображение требуется произвести ряд вычислений, соответственно, сложные изображения могут требовать повышенных вычислительных мощностей.
  • Не каждая графическая сцена может быть представлена в векторном виде: для сложного изображения с широкой цветовой гаммой может потребоваться огромное количество точек и кривых, что нивелирует все преимущества векторной графики.

Самые популярные векторные форматы: SVG, AI.
