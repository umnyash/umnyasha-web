Метод createImageData.

Метод createImageData создаёт новый объект ImageData с заданными размерами. Цвет пикселей будет назначен как прозрачный чёрный (rgba(0, 0, 0, 0)).

  • createImageData(width, height) — создать ImageData c шириной width и высотой height.
  • createImageData(imageData) — создать ImageData с шириной и высотой, как у переданного объекта ImageData. Само изображение не копируется.
  • createImageData(width, height, settings) — создать ImageData c шириной width и высотой height и с заданными настройками settings.

Настройки settings представлены объектом с единственным свойством — colorSpace, которое задаёт пространство цветов.

Значения:

  • 'srgb' — (по умолчанию).
  • 'display-p3'.
