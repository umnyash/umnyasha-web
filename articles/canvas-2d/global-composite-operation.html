Свойство globalCompositeOperation.

Свойство globalCompositeOperation — устанавливает тип операции композиции, применяемой при рисовании новых фигур.

Значения:

  • 'source-over' — (по умолчанию) новая фигура рисуется поверх имеющегося содержимого холста.
  • 'source-in' — новая фигура рисуется только там, где она перекрывается с содержимым холста, а всё остальное становится прозрачным. Иными словами новая фигура превращает имеющееся содержимое холста в свою маску.
  • 'source-out' — новая фигура рисуется только там, где она не перекрывается с содержимым холста, при этом прежнее содержимое холста становится прозрачным.
  • 'source-atop' — новая фигура рисуется только там, где она перекрывается с содержимым холста.
  • 'destination-over' — новая фигура рисуется позади имеющегося содержимого холста.
  • 'destination-in' — имеющееся содержимое холста сохраняется там, где перекрвается новой фигурой, а всё остальное становится прозрачным. Иными словами фигура сработала как маска для имеющегося содержимого холста.
  • 'destination-out' — имеющееся содержимое холста сохраняется там, где оно не перекрывается новой фигурой. Иными словами, новая фигура делает очищает область холста, соответсвующую своей фигуре.
  • 'destination-atop' — имеющееся содержимое холста сохраняется там, где перекрвается с новой фигурой, а новая фигура рисуется позади.

Значения связанные со смешиванием цветов:

  • 'lighter'.
  • 'copy'.
  • 'xor'.
  • 'multiply'.
  • 'screen'.
  • 'overlay'.
  • 'darken'.
  • 'lighten'.
  • 'color-dodge'.
  • 'color-burn'.
  • 'hard-light'.
  • 'soft-light'.
