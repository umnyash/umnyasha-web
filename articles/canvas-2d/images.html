Использование изображений

Источники изображения:

  • HTMLImageElement — изображения, созданные с помощью конструктора Image(), а также любого элемента <img>.
  • SVGImageElement — изображения встроенные с помощью <image> (этот тег используется для встраивания svg и растровых изображений внутрь тега <svg></svg>).
  • HTMLVideoElement — использование HTML-элемента <video></video> в качестве источника изображения захватывает текущий кадр из видео и использует его в качестве изображения.
  • HTMLCanvasElement — другой Canvas.
  • ImageBitmap — обрезанное растровое изобржение. Такой тип используется для извлечения изображения из спрайта.
  • OffscreenCanvas — особый вид Canvas, который не отображается и подготавливается без отображения.
  • VideoFrame — изображение, представляющее один кадр видео.

Отрисовка изображений:

  • drawImage — отрисовать изображение.

Сглаживание изображений:

  • imageSmoothingEnabled — сглаживание изображений при масштабировании.
  • imageSmoothingQuality — качество сглаживания изображений.
