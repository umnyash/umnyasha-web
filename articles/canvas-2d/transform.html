Метод transform

Метод transform(a, b, c, d, e, f) — умножить текущую трансформацию на матрицу, описанную аргументами.

  • a — (m11) масштабирование по оси X.
  • b — (m12) наклон по оси X.
  • c — (m21) наклон по оси Y.
  • d — (m22) масштабирование по оси Y.
  • e — (dx) перемещение по оси X.
  • f — (dy) перемещение по оси Y.

Это позволяет масштабировать, вращать, переводить (перемещать) и наклонять сетку координат.
