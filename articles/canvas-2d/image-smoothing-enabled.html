Свойство imageSmoothingEnabled.

Свойство imageSmoothingEnabled — сглаживание изображений при масштабировании.

Значения:

  • true — (по умолчанию) включено.
  • false — отключено.

Это свойство полезно для игр и других приложений, использующих пиксельную графику. При увеличении изображений алгоритм изменения размера по умолчанию размывает пиксели. Установите значение false, чтобы сохранить резкость пикселей.
