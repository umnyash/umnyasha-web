Метод toDataURL.

Метод холста toDataURL(type, encoderOptions) — возвращает URL-адрес данных, содержащий представление изображения в указанном формате type.

type — необязательный аргумент, задаёт тип изображения.

Значения:

  • 'image/png' — (по умолчанию).
  • 'image/jpg'.
  • 'image/webp'.

encoderOptions — необязательный аргумент, задаёт качество изображения, которое будет использоваться при создании изображений в форматах, поддерживающих сжатие с потерями, например, JPG или WebP.

Значения:

  • Число от 0 до 1.

Значение по умолчанию используется, если аргумент не передан или некорректен.
