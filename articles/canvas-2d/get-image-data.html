Метод getImageData.

Метод getImageData(left, top, width, height) — вернуть объект ImageData с данными о пикселях для прямоугольной области холста шириной width и высотой height, верхний левый угол которой расположен в позиции left top.

Если область выходит за пределы холста, пикселям за пределами холста задаётся прозрачный чёрный цвет (rgba(0, 0, 0, 0)).
