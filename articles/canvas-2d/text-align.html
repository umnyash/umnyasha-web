Свойство textAlign

Свойство textAlign — выравнивание текста относительно своей позиции x.

  • left — текст располагается справа от x.
  • right — текст располагается слева от x.
  • center — текст центрируется относительно x.
  • start — (значение по умолчанию) текст располагается справа от x для языков с письмом слева-направо, а для языков с письмом справа-налево текст располагается слева от x.
  • end — как start только наоборот.
