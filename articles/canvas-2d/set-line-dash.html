Метод setLineDash.

Метод setLineDash(segments) — установить паттерн прерывистости обводки.

  • segments — массив чисел, поочередно задающих ширину линии и ширину разрыва. Если количество элементов в массиве нечётное, элементы массива копируются и объединяются. Например, [5, 15, 25] станет [5, 15, 25, 5, 15, 25]. Значением по умолчанию является пустой массив.

Получить заданный паттерн можно с помощью метода getLineDash.
