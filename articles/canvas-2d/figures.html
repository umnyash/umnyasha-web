Создание фигур.

В отличие от SVG, Canvas поддерживает только одну примитивную фигуру: прямоугольник. Все другие фигуры должны быть созданы комбинацией одного или большего количества путей (наборов точек, соединённых линиями).

Создание фигур с помощью путей.

Создание фигур используя пути происходит в несколько шагов:

  1. Открытие нового пути.
  2. Создание подпутей, которые будут являться частями пути и сформируют фигуру.
  3. Обводка и/или заливка пути, чтобы отобразить фигуру.

Новый путь.

  • beginPath — начать новый путь.

Методы создания подпутей.

  • moveTo — переместить перо.
  • lineTo — добавить линию.
  • closePath — закрыть путь.
  • rect — добавить прямоугольник.
  • arc — добавить дугу вдоль окружности.
  • arcTo — добавить дугу, скругляющую угол.
  • quadraticCurveTo — добавить квадратичную кривую Безье.
  • bezierCurveTo — добавить кубическую кривую Безье.

Отображение пути.

  • stroke — обвести текущий путь.
  • fill — заполнить текущий путь.

Рисование прямоугольников.

  • fillRect — нарисовать заполненный прямоугольник.
  • strokeRect — нарисовать обведённый прямоугольник.
  • clearRect — очистить прямоугольную область.

Обрезка.

  • clip — закрыть путь и сделать из него контур обрезки холста.

Контур обрезки похож на обычную фигуру, но он действует как маска. Все, что выходит за пределы этой фигуры, не будет нарисовано на холсте.

По умолчанию Canvas имеет контур обрезки того же размера, что и сам холст, и никакого отсечения не происходит.

Композиция

По умолчанию фигуры всегда рисуются одна поверх другой. Это поведение можно изменить.

  • globalCompositeOperation — устанавливает тип операции композиции, применяемой при рисовании новых фигур.
