Манипуляции с пикселями.

Объект imageData содержит пиксельные данные об участке холста.

Свойства объекта:

  • width — ширина участка холста.
  • height — высота участка холста.
  • data — Uint8ClampedArray. Одномерный массив содержащий RGBA-данные о пикселях с целочисленными значениями от 0 до 255 включительно. Информация о пикселях хранится по их порядке слева направо и сверху вниз.

Использование imageData.

  • createImageData — создать новый объект imageData.
  • getImageData — вернуть imageData с данными о пикселях холста.
  • putImageData — расположить на холсте пиксели из imageData.

Холст можно сохранить в качестве изображения с помощью метода холста — toDataURL.
