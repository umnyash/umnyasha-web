Метод addColorStop

Метод addColorStop(position, color) — добавить колорстоп.

  • postion — относительная позиция цвета в сердце градиента. Принимает значения от 0 (начало) до 1 (конец).
  • color — цвет в любом CSS-формате.

Метод вызывается у созданного градиента.

Пример:

const gradient = ctx.createLinearGradient(0, 0, 0, 100);
gradient.addColorStop(0, 'black');
gradient.addColorStop(0.5, 'pink');
gradient.addColorStop(0.5, 'red');
gradient.addColorStop(1, 'green');
