fetch

С помощью функции fetch можно отправлять сетевые запросы на сервер — получать и отправлять данные.

fetch(url[, options]);
  1. url — адрес, по которому нужно сделать запрос.
  2. options — объект конфигурации, в котором можно настроить метод запроса, тело запроса, заголовки и многое другое.

Метод возвращает промис с объектом ответа, содержащем дополнительную информацию (статус ответа, заголовки) и ответ на запрос.
