async/await

Оператор async позволяет определить асинхронную функцию.

async function foo() {}          // Function Declaration
const foo = async function() {}  // Function Expression
const foo = async () => {}       // Arrow Function Expression

Асинхронная функция всегда возвращает промис (если возвращаемое значение не промис, то оборачивается в промис).

Асинхронная функция может содержать оператор await, который приостанавливает выполнение асинхронной функции и ожидает завершения переданного промиса. Если промис выполняется, то возращает его результат, иначе генерирует исключение.
