Производительность вёрстки.

Поисковики отдают предпочтение быстрым сайтам.

Из чего состоит загрузка сайта.

  1. Получение IP-адреса по названию домена (DNS).
  2. Подключение к серверу по IP-адресу.
  3. Установление защищённого соединения (HTTPS).
  4. Запрос HTML-страницы и ожидание ответа.
  1. Загрузка HTML-страницы.
  2. Парсинг HTML-документа и построение очереди загрузки ресурсов.
  3. Загрузка и парсинг CSS-стилей.
  4. Загрузка и отображение веб-шрифтов.
  5. Загрузка и выполнение JavaScript-кода.
  6. Загрузка изображений.
  7. Завершение рендеринга страницы и выполнение отложенного JavaScript-кода.

Приоритеты загрузки.

  1. Highest.
  2. High.
  3. Medium.
  4. Low.
  5. Lowest.

Предзагрузка ресурсов.

<head>
  <link rel="preload" href="/css/style.min.css" as="style">
  <link rel="preload" href="/js/script.min.js" as="script">
  <link rel="preload" href="/img/bg-header@2x.jpg" as="image">
  <link rel="preload" href="/fonts/psans-narrow.woff2" as="font" type="font/woff2" crossorigin>
</head>

Ленивая загрузка для img и iframe.

<img src="files/cat.jpg" loading="lazy" alt="Небольшой серый кот.">
